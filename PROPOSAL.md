# [DAKER World Cup Manager Tactics Web Challenge] 기획서
**서비스명: K-Tactics Lab 2026**
**부제: 2026 월드컵 대한민국 국가대표팀 전술 솔루션 및 인터랙티브 감독 시뮬레이터**

---

## 1. 서비스 개요 (Service Overview)

### 1.1 기획 배경 및 문제 제기
2026년 국제축구연맹(FIFA) 북중미 월드컵을 앞두고 대한민국 축구 국가대표팀에 대한 국민적 관심이 집중되고 있습니다. 그러나 최근 대표팀의 주요 경기력과 전술에 대하여 축구 팬 및 전문가들 사이에서는 다음과 같은 핵심 전술적 과제들이 지속적으로 제기되고 있습니다.

1. **단조로운 빌드업 패턴:** 상대의 밀집 수비 시 중원 정면 돌파나 하프스페이스 활용 대신, 최후방 수비선에서 횡패스와 백패스만을 반복하는 구조적 문제(U자형 빌드업 논란).
2. **측면 수비(풀백) 밸런스 불안:** 공격 가담 시 발생한 뒷공간을 신속히 커버하지 못해 상대의 역습에 취약해지는 구조적 밸런스 불균형.
3. **개인 전술 의존도 심화:** 체계적인 팀 전술과 플랜 B의 부재로 인해 후반전 체력 저하 시 특정 주전 스타 선수의 개인 역량에만 의존하는 현상.

기존의 축구 미디어 및 커뮤니티 환경에서는 일방적인 관람과 텍스트 기반의 비평만이 이루어졌으며, 전문 전술 시뮬레이션 게임(Football Manager 등)은 일반 대중이 접근하기에 높은 진입장벽과 복잡성을 지니고 있습니다. 이에 따라 일반 대중이 직관적이고 가벼운 인터랙션을 통해 대표팀의 전술적 과제를 이해하고 직접 해결해 볼 수 있는 인터랙티브 웹 플랫폼의 필요성이 대두되었습니다.

### 1.2 서비스 정의
**K-Tactics Lab 2026**은 웹 브라우저 상에서 별도의 설치나 회원가입 절차 없이, 사용자가 직접 대한민국 축구 국가대표팀의 수석 감독이 되어 핵심 전술 과제를 설정 및 해결하고, 그 결과에 따른 실시간 대중 여론 지지도(Vibe Meter) 변화를 정량적으로 시뮬레이션하는 **대중 참여형 인터랙티브 전술 시뮬레이션 웹 서비스**입니다.

### 1.3 서비스의 독창성 및 차별점
1. **즉각적인 접근성 (Zero-Setup & Lightweight):** Vanilla HTML5, CSS3, JavaScript 기반으로 구현되어 모바일 및 데스크톱 환경에서 초고속 로딩을 보장하며, 심사위원 및 일반 사용자의 접근 장벽을 완전히 제거했습니다.
2. **디바이스 맞춤형 조작 체계 (Hybrid Interaction):** 데스크톱 환경에서는 직관적인 마우스 드래그 앤 드롭(Drag & Drop)을, 모바일 및 터치스크린 환경에서는 클릭 기반의 선택 및 맞교환(Click-to-Swap) 인터랙션을 지원하여 어떠한 기기에서도 완벽한 조작성을 보장합니다.
3. **실시간 여론 시뮬레이션 (Dynamic Vibe Meter & Coaching AI):** 감독의 결정에 따라 대중 여론 지지율이 실시간으로 연동되며, AI 코치가 전술적 타당성을 평가하여 피드백을 제시하는 게임적 동기부여 메커니즘을 적용했습니다.
4. **바이럴 촉진 메커니즘 (Viral Trigger Mechanism):** 시뮬레이션 완료 후 생성되는 최종 감독 성적 및 전술 명함을 고화질 PNG 이미지로 추출하여, 사용자의 자발적인 소셜 미디어 인증 및 커뮤니티 공유를 유도하도록 설계했습니다.

## 2. "감독 경험" 설계 의도 (Design Intent of "Manager Experience")

### 2.1 감독 경험의 핵심 개념
현대 축구에서 국가대표팀 감독의 의사결정은 단순한 승패 통계를 넘어, 국가적 기대감과 대중 여론(Public Sentiment)을 좌우하는 핵심 요인입니다. 본 서비스는 사용자가 감독석에 앉아 **"전술적 선택과 스쿼드 배치가 팀 밸런스 및 실시간 대중 여론 지지율에 어떠한 상충 관계(Trade-off)를 발생시키는가"**를 직접 체감하도록 설계되었습니다.

### 2.2 3대 핵심 과제를 통한 의사결정 체험
사용자는 대표팀이 직면한 3가지 핵심 전술 과제를 단계적으로 해결하며 전략적 의사결정을 수행합니다.

1. **과제 1: 빌드업 루트 및 탈압박 설정**
   * **설계 의도:** 무의미한 백패스를 제한하고 하프스페이스(중앙과 측면 사이의 공간) 침투 지침을 활성화할 경우, 공격 파괴력과 대중의 여론 지지율이 급상승하는 긍정적 피드백을 제공합니다. 반면 지침을 해제할 경우 단조로운 공격에 대한 코칭스태프의 우려 메시지가 표출됩니다.
2. **과제 2: 풀백(윙백) 전술 밸런스 설정**
   * **설계 의도:** ▲인버티드 풀백(중원 빌드업 가담 및 수적 우위 확보), ▲쓰리백 스토퍼(측면 수비 강화 및 실점 리스크 최소화), ▲오버래핑 윙백(측면 크로스 공격 극대화) 중 하나를 선택하게 합니다. 예를 들어 오버래핑 올인 선택 시 공격력은 상승하지만 후반전 체력 유지력이 저하되는 수치적 상충을 구현했습니다.
3. **과제 3: 후반전 플랜 B 조커 투입**
   * **설계 의도:** 경기 60분 이후 주전 선수의 체력이 떨어졌을 때를 대비하여, 오현규(피지컬 타겟맨), 배준호(크랙 드리블러), 양민혁(고속 역습), 주민규(박스 안 결정력) 중 상대 수비 전술을 공략할 최적의 교체 선수를 선택하게 하여 감독의 위기관리 능력을 시험합니다.

## 3. 페이지 구성 및 UI 레이아웃 (Page Structure)

본 서비스는 페이지 간 이동 시 발생하는 몰입도 저하를 방지하기 위해, 한 화면 내에서 전술 설정, 실시간 피드백, 시뮬레이션 결과 확인이 유기적으로 이루어지는 **'원페이지 3단 그리드 레이아웃(One-Page 3-Column Grid Layout)'**으로 구현되었습니다.

```text
+-----------------------------------------------------------------------------------+
|  [Header Bar] K-Tactics Lab 2026 | 팬 여론 지지도: 92% | 현재 포메이션: 4-2-3-1    |
+------------------------------+------------------------------+---------------------+
|  [좌측 패널: 여론 및 코치]     |  [중앙 패널: 전술판 및 스쿼드]|  [우측 패널: 진단]  |
|                              |                              |                     |
|  [실시간 여론 지지도 바]     |  [포메이션 선택 컨트롤러]    |  [팀 밸런스 진단]   |
|  - 지지율 수치 및 상태 문구  |  - 4-3-3 / 3-5-2             |  - 공격 파괴력      |
|                              |  - 4-2-3-1 / 4-4-2           |  - 측면 수비 안정도 |
|  [AI 수석 코치 대화 창]      |                              |  - 중원 장악력      |
|  - 전술 변경 피드백 및 경고  |  [인터랙티브 축구장 전술판]  |  - 체력 유지력      |
|                              |  - 11명 포지션 그리드        |                     |
|  [실시간 네티즌 중계창]      |  - 드래그/클릭 하이브리드 존 |  [과제 2: 풀백 설정]|
|  - 지지율 연동 댓글 스트리밍 |                              |  - 인버티드 풀백    |
|                              |  [교체 대기 벤치 멤버]       |  - 쓰리백 스토퍼    |
|  [과제 1: 빌드업 지침 설정]  |  - 5명 후보 선수 슬롯        |  - 오버래핑 윙백    |
|  - 하프스페이스 침투 토글    |  - 구장 선수와 교체 가능     |                     |
|  - 백패스 금지 토글          |                              |  [과제 3: 조커 선택]|
|  - 이강인 프리롤 토글        |                              |  - 오현규 / 배준호  |
|                              |                              |  - 양민혁 / 주민규  |
|                              |                              |                     |
|                              |                              |  [시뮬레이션 가동]  |
+------------------------------+------------------------------+---------------------+
```

### 3.1 영역별 구성 명세
1. **상단 헤더 (Header Bar):** 프로젝트 타이틀, 실시간 대중 여론 지지율 수치, 현재 선택된 포메이션 상태를 표시합니다.
2. **좌측 패널 (Left Panel - Public Sentiment & Coaching AI):**
   * 실시간 여론 지지도(Vibe Meter): 0~100% 진행 바 및 현재 여론 상태를 요약한 텍스트 출력.
   * AI 수석 코치 대화 창: 유저의 전술 조작에 반응하여 분석 및 경고 문구를 출력하는 코칭 위젯.
   * 실시간 네티즌 중계창: 현재 여론 지지율 구간에 맞추어 대중 반응 댓글이 자동 스크롤되는 스트리밍 위젯.
   * 과제 1 컨트롤러: 빌드업 및 탈압박 전술을 켜고 끌 수 있는 토글 스위치 3종.
3. **중앙 패널 (Center Panel - Interactive Pitch):**
   * 포메이션 선택 바: 4종의 대표 포메이션(4-3-3, 3-5-2, 4-2-3-1, 4-4-2) 즉시 전환 버튼.
   * 축구장 전술판: 11명의 선수가 포지션별(공격-중원-수비-골키퍼)로 배치되는 영역이며, 드래그 앤 드롭 및 클릭 선택을 이원 지원.
   * 벤치 스쿼드 영역: 교체 후보 선수 5명이 배치되며, 전술판 내의 선수와 자유롭게 맞교환 가능.
4. **우측 패널 (Right Panel - Team Balance & Strategy):**
   * 팀 밸런스 진단 바: 공격, 수비, 중원, 체력 4대 속성의 정량적 수치(0~100) 시각화.
   * 과제 2 컨트롤러: 풀백의 전술적 임무를 지정하는 단일 선택(Radio-style) 버튼 3종.
   * 과제 3 컨트롤러: 후반전 교체 투입을 위한 조커 선수 선택 카드 4종.
   * 시뮬레이션 가동 버튼: 최종 시뮬레이션 및 결과 보고서 모달을 호출하는 메인 액션 버튼.
5. **오버레이 모달 (Modals):**
   * 선수 임무 설정 모달: 선수 카드 클릭 시 호출되며, 상세 역할(Role)을 지정하는 창.
   * 최종 리포트 모달: 3단계 조별리그 매치 중계 애니메이션 및 고화질 명함 카드 PNG 다운로드 기능을 제공하는 창.

## 4. 핵심 인터랙션 명세 (Core Interaction Specs)

### 4.1 인터랙션 A: 디바이스 이원화 스쿼드 배치 및 교체 (Hybrid Squad Builder)
* **동작 명세:** 모바일 터치스크린 환경에서의 HTML5 드래그 이벤트 제약을 극복하고 완벽한 호환성을 제공하기 위해 **이원화된 인터랙션 체계**를 구현했습니다.
  * **PC 데스크톱 환경:** HTML5 Drag & Drop API를 적용하여, 선수 카드를 마우스로 잡고 다른 슬롯이나 하단 벤치 멤버와 드래그하여 직관적으로 맞바꿉니다.
  * **모바일 및 터치 환경:** 클릭 기반 맞교환(Click-to-Swap) 방식을 적용하여, 교체할 선수를 터치로 선택한 후 타겟 슬롯을 터치하면 즉시 포지션이 교환됩니다.
* **상태 변화 및 시각적 피드백:**
  * 선택/드래그 시작: 대상 카드의 투명도를 40%로 낮추고(`opacity: 0.4`), 활성화 상태 테두리를 강조합니다.
  * 타겟 진입/지정: 타겟 슬롯이 1.12배 확대(`scale: 1.12`)되고 에메랄드빛 테두리가 발광하여 교체 가능 영역임을 직관적으로 안내합니다.

### 4.2 인터랙션 B: 비정상 배치 감지 및 AI 코치 경고 시스템
* **동작 명세:** 선수 교체 시 시스템은 배치된 선수의 고유 포지션 속성과 현재 위치를 비교하는 조건문 로직을 구동합니다.
  * 예시 1: 대표적인 공격수(손흥민 등)가 최후방 수비수(CB, LB, RB) 또는 골키퍼(GK) 슬롯에 배치되는 경우.
  * 예시 2: 골키퍼(조현우 등)가 최전방 공격수(ST, LW, RW) 슬롯에 배치되는 경우.
* **피드백 연출:**
  * AI 코치 경고: 코치 대화 창에 *"AI 코치 비상 경보: 공격수를 최후방에 배치하는 전례 없는 기행입니다!"* 등의 경고 문구가 표출됩니다.
  * 수치 패널티: 대중 여론 지지율(Vibe Score)이 즉시 25%~30% 감소합니다.
  * 시각/감정적 연출: 지지율이 30% 이하로 저하될 경우, 화면 전체가 좌우로 미세하게 진동하는 **CSS Shake 애니메이션**과 붉은색 경고 점멸 효과가 가동되며, 네티즌 중계창에 강한 비판 댓글이 표출됩니다.

### 4.3 인터랙션 C: 클릭 기반 선수 세부 임무(Role) 설정
* **동작 명세:** 축구장 위에 배치된 선수 카드를 마우스 좌클릭(모바일은 더블 터치 또는 버튼 활용)하면, 해당 선수의 포지션 대분류(공격/중원/수비/골키퍼)에 매칭되는 3~5개의 전술 임무 리스트(예: 인사이드 포워드, 딥라잉 플레이메이커, 커버링 센터백 등)가 포함된 팝업 모달이 나타납니다.
* **피드백:** 특정 임무를 선택하면 모달이 닫히면서 해당 선수의 하단 태그에 새 임무가 반영되며, 팀 밸런스 점수 및 여론 지지율이 소폭 상승(+3%)합니다.

### 4.4 인터랙션 D: 시뮬레이션 애니메이션 및 PNG 이미지 캡처
* **동작 명세:** 우측 하단의 '시뮬레이션 가동' 버튼 클릭 시 최종 리포트 모달이 나타나며, 조별리그 1차전, 2차전, 3차전의 가상 매치 중계 텍스트가 1.4초 간격으로 애니메이션 렌더링됩니다.
* **명함 추출 기능:** 중계 완료 후 나타나는 최종 감독 평가 카드(예상 성적, 지지율, 감독 스타일 타이틀 포함)에서 **'명함 카드 PNG 다운로드'** 버튼을 누르면, `html2canvas` 라이브러리가 해당 DOM 영역을 2배수 해상도(`scale: 2`)의 캔버스로 변환하여 즉시 클라이언트 환경에 PNG 이미지 파일로 다운로드합니다.

## 5. 데이터 활용 방식 및 알고리즘 (Data Usage & Simulation Logic)

### 5.1 경량화된 게임 밸런스 보정 알고리즘 (Lightweight Game Balance Modifier)
본 서비스는 복잡한 물리 엔진이나 예측 AI 대신, 대중이 전술 선택에 따른 상충 관계(Trade-off)를 즉각적이고 직관적으로 체감할 수 있도록 최적화된 **'경량화 게임 밸런스 보정 알고리즘'**을 채택했습니다. 4대 속성(공격 파괴력, 측면 수비 안정도, 중원 장악력, 체력 유지력)은 포메이션 기본값에 전술 토글 및 풀백 역할 보정값이 선형 합산되어 연산됩니다.

$$\text{Stat}_{\text{type}} = \text{BaseStat}(\text{Formation})_{\text{type}} + \sum \Delta\text{Tactic}_{\text{type}} + \sum \Delta\text{Role}_{\text{type}}$$

* **포메이션별 기본값 산정:** 
  * 3-5-2 선택 시: 수비 안정도 85, 중원 장악력 88, 공격 파괴력 72 (중원 밀집 및 수비 강화)
  * 4-2-3-1 선택 시: 공격 파괴력 88, 중원 장악력 84, 수비 안정도 70 (공격 코어 중심)
* **상충 관계(Trade-off) 적용 예시:**
  * '풀백 오버래핑 윙백' 선택 시: 공격 파괴력 +15 증가, 수비 안정도 -15 감소, 후반전 체력 유지력 -20 감소.
  * '풀백 쓰리백 스토퍼' 선택 시: 수비 안정도 +25 증가, 공격 파괴력 -10 감소.

### 5.2 대중 여론 지지도(Vibe Score) 연동 메커니즘
여론 지지도는 기본값 50%에서 시작하며, 사용자의 선택이 대중의 기대에 부합하는지에 따라 정량적으로 변동합니다. 최종 값은 10%에서 100% 사이로 제한(Clamping)됩니다.

$$\text{Vibe Score} = \max\left(10, \min\left(100, 50 + \sum \text{Positive Modifiers} - \sum \text{Negative Modifiers}\right)\right)$$

* **긍정 요인:** 무의미한 백패스 금지 활성화(+18%), 하프스페이스 침투 활성화(+12%), 이강인 프리롤 부여(+10%), 적절한 조커 선정(+5%).
* **부정 요인:** 비정상 포지션 배치(-25%~-30%), 체력 고갈을 유발하는 극단적 전술 선택(-5%).
* **시각 상태 변동:**
  * **80% 이상 (최고 지지):** 에메랄드색 테두리 발광 효과, 축하 메시지 및 긍정적 라이브 댓글 출력.
  * **40% ~ 79% (안정/관망):** 시안색 및 앰버색 상태 표시, 중립적 피드백 유지.
  * **39% 이하 (위기/분노):** 붉은색 텍스트 전환, 화면 진동(Shake) 효과 및 강한 비판 댓글 출력.

### 5.3 네티즌 중계창 스트리밍 데이터 구조
현장감 있는 대중 반응 구현을 위해, 시스템은 4.5초 주기(`setInterval`)로 현재 여론 지지도 구간에 매칭되는 댓글 풀(Pool)에서 임의의 데이터를 추출하여 채팅 영역에 동적으로 추가하고 스크롤 위치를 최하단으로 유지합니다.
* **데이터 구조:** `{ user: '사용자ID', text: '댓글내용', type: 'normal' | 'vip' | 'hater' }`

## 6. 주요 사용 흐름 (Main User Flow - Journey Map)

```text
[단계 1: 서비스 진입 및 탐색]
   │
   ├─► 웹 브라우저 접속 (별도 설치/로그인 없음)
   └─► 초기 상태 확인: 포메이션 4-3-3, 여론 지지율 50%, AI 코치 안내 문구 확인
           │
           ▼
[단계 2: 전술 과제 설정 및 즉각적 피드백 체험]
   │
   ├─► 과제 1 해결: '무의미한 U자형 백패스 금지' 토글 활성화
   │     └─► (결과) 여론 지지도 +20% 상승, 코치 긍정 피드백, 중계창 환호 댓글 출력
   │
   ├─► 과제 2 해결: 풀백 역할을 '쓰리백 스토퍼'로 변경
   │     └─► (결과) 측면 수비 안정도 +25 상승, 공격 파괴력 -10 감소 (밸런스 확인)
   │
   └─► 과제 3 해결: 후반전 플랜 B 조커로 '오현규' 선수 선택
         └─► (결과) 여론 지지도 +5% 추가 상승
           │
           ▼
[단계 3: 스쿼드 조율 및 의사결정 실험]
   │
   ├─► 마우스 드래그(또는 클릭 선택)로 벤치 선수(이동경)와 구장 선수 교체
   │
   ├─► 호기심에 의한 의사결정 실험: 공격수(손흥민)를 수비수 슬롯으로 드래그
   │     └─► (결과) AI 코치 경고 표출, 지지율 -25% 폭락, 화면 진동(Shake) 효과 발생
   │     └─► (복구) 선수를 원래 공격수 위치로 원상 복구하여 지지율 회복
   │
   └─► 선수 카드 클릭 -> 세부 임무(Role) '인사이드 포워드' 설정
           │
           ▼
[단계 4: 시뮬레이션 가동 및 최종 평가]
   │
   ├─► '2026 월드컵 시뮬레이션 가동' 버튼 클릭
   ├─► 조별리그 1차전, 2차전, 3차전 경기 전개 텍스트 애니메이션 시청 (약 4.2초)
   │
   └─► 최종 리포트 확인: 예상 성적(8강 진출), 최종 지지율(92%), 감독 스타일 칭호 확인
           │
           ▼
[단계 5: 결과 소장 및 바이럴 공유]
   │
   ├─► '명함 카드 PNG 다운로드' 버튼 클릭
   └─► 생성된 이미지(`K-Tactics_2026_내감독명함.png`)를 소셜 미디어 및 커뮤니티에 공유
```

1. **진입 단계:** 사용자는 접속 즉시 전체 인터페이스와 대중 여론 지지율 50%를 확인합니다.
2. **전술 설정 단계:** 한국 축구의 문제점인 '백패스 금지'를 선택하여 대중 지지율이 상승하는 긍정적 보상을 체감합니다.
3. **스쿼드 실험 단계:** 드래그 앤 드롭(또는 클릭 교체)으로 선수를 조율하며, 비정상 배치 시 발생하는 AI 코치의 경고와 화면 진동을 통해 시스템의 인과관계를 경험합니다.
4. **시뮬레이션 단계:** 설정한 전술에 따른 조별리그 경기 결과와 최종 성적표를 확인합니다.
5. **공유 단계:** 최종 평가 명함 카드를 고화질 PNG 이미지로 저장하여 개인 소셜 미디어 및 해커톤 바이럴에 활용합니다.
