# K-Tactics Lab 2026 — 사용 가이드 & 작동 원리

> 이 앱이 무엇이고, 어떻게 플레이하며, 각 기능이 왜 존재하는지 정리한 문서다.
> 모든 설명은 실제 코드(`app.js`, `index.html`, `data/`, `api/coach.js`)에 근거한다.

---

## 0. 한 줄 요약

당신이 대한민국 대표팀 **감독**이 되어, 실제로 있었던 **2026 월드컵 남아공전 0-1 패(조별리그 탈락)를 다시 지휘**해 결과를 바꾸는 인터랙티브 전술 시뮬레이터다.

---

## 1. 이 앱이 하려는 것 (컨셉)

**배경(실화):** 2026 북중미 월드컵에서 한국은 조별리그 최종전 남아공에 0-1로 졌고, 언론은 3대 전술 문제를 집중 비판했다.

1. **U자형 빌드업** — 중앙 침투 없이 백패스만 반복하는 단조로운 공격
2. **풀백 자동문** — 측면 뒷공간이 뚫리는 수비 불안
3. **해줘 축구** — 체계 없이 손흥민·이강인 개인 기량에만 의존

이 앱은 그 3대 문제를 **"네가 감독이라면 어떻게 고칠래?"** 로 바꿔, 전술 다이얼·라인업·교체로 직접 풀게 한다.

**핵심 재미 4가지**

- 내 전술에 따라 **실제 승률이 바뀌는 걸 1,000회 몬테카를로로** 확인
- **AI 상대 감독**과의 수싸움 (상대가 내 라인업에 맞춰 카운터를 짬)
- **대중 여론**이 내 선택에 실시간 반응
- 결과를 **감독 명함**으로 만들어 공유하고 친구에게 도전장 보내기

---

## 2. 목표 (무엇을 달성하면 되나)

- 그날의 0-1 패를 뒤집어 **승리 또는 무승부(무승부 시 PK 승)** 를 만든다.
- 동시에 **팬 여론(지지율)** 을 높여 좋은 감독 타이틀을 얻는다.
- **중요:** 승패와 여론은 별개다. 여론이 높다고 이기는 게 아니다. 여론은 "대중이 널 얼마나 좋아하나"를 보여주는 **스토리 지표**이고, 실제 승패는 아래 7장의 몬테카를로가 결정한다.

---

## 3. 화면 구성 (지금 뭘 보고 있는 건지)

상단에 탭 2개가 있다: **[⚽ 전술 지휘 보드]** 와 **[🚪 선수단 락커룸]**.

### 3.1 전술 지휘 보드 (메인 화면, 3단)

| 위치 | 이름 | 무엇 |
| :-- | :-- | :-- |
| **좌측** | 팀 밸런스 진단 | 공격 파괴력 / 측면 수비 안정도 / 중원 장악 / 체력 4대 스탯 (FBref 기반). 내 전술의 "성적표" |
| 좌측 하단 | 팬 네티즌 중계창 | 여론 지지율 바 + 상황에 맞는 댓글이 4.5초마다 자동 스트리밍 |
| **중앙 상단** | 매치 컨트롤 헤더 | 포메이션 4종 + 상대국 3종 선택, 경기 국면 상태 |
| 중앙 | 축구장 전술판 | 선발 11명 배치. 드래그/클릭으로 교체 |
| 중앙 하단 | 하단 탭 | [벤치 9명] 과 [4대 전술 다이얼 콘솔] 을 전환 |
| **우측** | Coach V 채팅 | AI 수석 코치. 질문하거나 프리셋 버튼으로 조언 요청 |
| 헤더 우상단 | 🚀 시뮬레이션 버튼 | 경기를 진행시키는 메인 버튼 (전반 → 후반으로 라벨이 바뀜) |

### 3.2 선수단 락커룸

2026 월드컵에 실제 출전한 **20인**의 카드(포지션·나이·출전 기록)를 조회하고, 카드를 클릭하면 FBref 기반 심층 스카우팅 리포트를 본다. (전술에는 영향 없는 열람용 자료실)

## 4. 플레이 흐름 (한 판을 처음부터 끝까지)

```text
[1] 진입 → 온보딩 모달
      "비기면 32강이었는데 남아공에 0-1 졌다"
      ├─ [⚽ 그 남아공전을 다시 지휘하기]  → 상대=남아공 자동 세팅
      └─ [자유 모드로 시작]              → 상대=멕시코, 자유롭게
                │
[2] 세팅 (킥오프 전) ── 여기서 대부분의 조작을 한다
      ├─ 포메이션 고르기 (4-3-3 등 4종)
      ├─ 상대국 고르기 (멕시코 / 스페인 / 남아공)
      ├─ 선수 교체·역할 지정 (드래그 or 클릭)
      └─ 4대 전술 다이얼 조정
          → 조작할 때마다 스탯·여론·Coach V·중계창이 즉시 반응
                │
[3] [🚀 시뮬레이션 가동] → 전반전 (0~45분)
      ├─ 이 순간 AI 상대 감독이 내 XI를 스카우팅해 카운터 전술 결정
      └─ 하프타임 스코어 산출 + 선수 체력 소모
                │
[4] 하프타임 (45분) ── 버튼이 [🔥 후반전 가동]으로 바뀜
      ├─ 선수별 체력 게이지 확인 (방전된 선수 체크)
      └─ 교체 예약: [60분 조커 투입] / [75분 닥공 전환]  또는 다이얼 재조정
                │
[5] [🔥 후반전 가동] → 1,000회 몬테카를로
      → 승 % · 무 % · 패 % + 가장 유력한 스코어 산출
                │
[6] 결과 분기
      ├─ 승 or 패 → 2D 하이라이트 → 감독 명함
      └─ 무승부   → PK 승부차기 (키커 5명 순서 지정) → 감독 명함
                │
[7] 감독 명함 카드
      감독 스타일 타이틀 · 예상 성적 · 여론 · 몬테카를로 분포
      └─ [명함 카드 저장] / [공유하기] / "이 전술 이겨봐" 도전 링크
```

## 5. 조작법 치트시트

| 하고 싶은 것 | 조작 | 결과 |
| :-- | :-- | :-- |
| 선수 위치 바꾸기 / 벤치와 교체 | PC: 카드 **드래그&드롭** · 모바일: 카드 클릭 → 대상 클릭(**맞교환**) | 스탯·여론 갱신 |
| 선수 세부 역할 바꾸기 | 카드를 **한 번 더 클릭** → 역할 모달 | 조직력 시너지 소폭 상승 |
| 포메이션 바꾸기 | 중앙 상단 포메이션 버튼 | 스탯 재계산 |
| 상대 바꾸기 | 중앙 상단 국기 버튼 | 상성·브리핑 변화 |
| 전술 지침 바꾸기 | 하단 [🕹️ 전술 콘솔] 탭 → 4대 다이얼 | Coach V 코멘트 + 여론 ↕ |
| AI 코치에게 묻기 | 우측 Coach V 입력창에 질문 | 실제 LLM 조언 (배포 시) |
| 경기 진행 | 헤더 [🚀 시뮬레이션] 버튼 (전반→후반) | 2단계 턴 진행 |

> **⚠️ 재미 포인트:** 손흥민 같은 공격수를 최후방 수비 슬롯에 놓으면 → 🚨 AI 코치 비상 경보 + 화면 진동 + 여론 급락. (일부러 해보면 반응을 볼 수 있다.)

## 6. 각 기능의 목적 / 의미

| 기능 | 왜 있나 (의미) |
| :-- | :-- |
| **4대 전술 다이얼** | 언론이 깐 3대 문제를 인터랙션으로 바꾼 핵심. 공격 루트=U자 빌드업 해결, 압박·성향=자동문/경기 운영 조율 |
| **선수 교체·역할** | 라인업 자유도. "손흥민을 선발로 되돌린다" 같은 실제 논란을 직접 재현 |
| **팀 밸런스 스탯** | 내 선택이 팀을 실제로 좋게 만들었는지 보여주는 정량 성적표 (FBref 기반) |
| **여론(Vibe) 미터** | 승패와 분리된 스토리 지표. "대중이 널 좋아했나" + 감독 타이틀 결정 |
| **Coach V (AI 코치)** | 내 보드 상태에 근거해 조언하는 조수. 배포 환경에선 실제 LLM |
| **AI 상대 감독** | "감독 대 감독"의 핵심. 내 라인업을 보고 상대 코치가 카운터를 짜서 시뮬에 반영 |
| **1,000회 몬테카를로** | 내 전술로 실제 승/무/패를 계산하는 진짜 확률 엔진 |
| **PK 승부차기** | 무승부 시 선수 침착성(composure) 기반 서브게임 |
| **감독 명함 / 공유 / 도전 링크** | 결과물 소장 + 바이럴. 내 전술을 URL로 인코딩해 친구에게 도전장 |

## 7. 엔진은 실제로 어떻게 작동하나 (정직한 내부 설명)

| 요소 | 실제 동작 |
| :-- | :-- |
| **선수 스탯** | 2026 WC 출전 20인의 FBref per-90 기록을 오프라인 스크립트(`scripts/parse_stats.py`)로 4대 능력치로 변환. 선발 11인 평균 + 포메이션·다이얼 보정 |
| **승패 (핵심)** | 후반전 양팀 득점을 **포아송 분포로 1,000회 표집**해 승/무/패 분포를 산출. 기대 득점 λ는 팀 스탯 ÷ 상대 강도 × 다이얼 × 체력 × AI 상대감독 카운터에서 유도. **결정론적이지 않고 실제 난수 기반이라 네 선택에 진짜로 반응한다** |
| **여론** | 승패와 완전히 분리된 별도 지표. 포메이션·다이얼·상성·라인업으로 계산 |
| **AI (Coach V / 상대 감독)** | 서버리스 프록시가 Groq → Gemini 순으로 LLM 호출, 실패 시 스크립트 폴백. 배포 URL에선 실제 LLM, `file://`(더블클릭)에선 스크립트로 자동 대체 |
| **오프라인 보장** | 인터넷/AI 없이 `index.html` 더블클릭만으로 전 기능 플레이 가능 (AI만 스크립트로 degrade) |

**직접 검증해보는 법:** 성향을 **닥공**으로 두고 시뮬 → 승률 확인. 다시 **잠그기**로 바꿔 시뮬 → 승률이 눈에 띄게 떨어진다. 상대를 **스페인**(강팀)으로 바꾸면 더 떨어진다. 숫자가 실제로 움직이면 엔진이 살아있다는 뜻.

## 8. 처음이라면: 권장 첫 플레이 5분

1. 온보딩에서 **[⚽ 그 남아공전을 다시 지휘하기]** 클릭
2. 축구장에 **손흥민이 선발(공격)** 로 있는지 확인 (벤치에 두면 여론 -18)
3. 하단 [🕹️ 전술 콘솔] 탭 → **공격 루트: U자 백패스 금지**, **압박: 중원 지역방어** 선택
4. 헤더 **[🚀 시뮬레이션 가동]** → 전반 종료까지 관전
5. 하프타임에서 **[60분 조커 투입 예약]** 클릭
6. **[🔥 후반전 가동]** → 승/무/패 % 와 최종 스코어 확인
7. **감독 명함**에서 [공유] 로 도전 링크 만들어보기

이 한 판을 돌리면 전체 루프가 한눈에 들어온다.
