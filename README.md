## 📌 1. Overview (서비스 개요)
**K-Tactics Lab 2026**은 대한민국 축구 국가대표팀이 2026 북중미 월드컵 본선 무대를 치르며 실전에서 직면한 핵심 전술적 과제들을 축구 팬(유저)이 직접 '국가대표 수석 감독'이 되어 해결해 보는 **대중 참여형 인터랙티브 전술 시뮬레이션 웹서비스**입니다. 

복잡한 설치나 회원가입 없이 브라우저에서 1초 만에 실행되며, 사용자의 전술적 선택(빌드업 지침, 풀백 역할, 조커 투입)과 스쿼드 배치에 따라 실시간 대중 여론 지지도(Vibe Meter)와 팀 밸런스가 유기적으로 출렁이는 게임적 카타르시스를 제공합니다.

## 🚨 2. Problem (문제 정의 및 언론 출처 근거)
**"기존 축구 콘텐츠는 일방적 비평만 있고, 전문 게임은 대중이 접근하기에 너무 무겁다."**

2026 북중미 월드컵 본선 실전 무대를 치르며, 국내외 주요 축구 매체(조선일보, 동아일보, 머니투데이, 경기일보 등)와 전문가들은 한국 축구를 향해 다음과 같은 **3대 고질적 전술 쟁점(Tactical Dilemmas)**을 집중적으로 보도하였습니다.
* **▲ U자형 빌드업 논란:** 중앙 침투 실종 및 의미 없는 횡·백패스 반복 (동아일보, 경기일보 등)
* **▲ 풀백 수비 자동문:** 오버래핑 시 뒷공간 노출 및 수비 조직력 붕괴 (조선일보, 머니투데이 등)
* **▲ 해줘 축구(스타 의존):** 체계적 팀 전술 부재와 손흥민·이강인 개인 역량 의존 (스포탈코리아, 뉴시스 등)

그러나 이를 소비하는 미디어 생태계에는 명확한 공백이 존재합니다.
* **기존 커뮤니티/댓글:** 언론이 지적한 문제를 일방적으로 성토할 뿐, 대중이 대안을 직접 설계하고 검증해 볼 맞춤형 도구가 없습니다.
* **전문 전술 게임(FM 등):** 현실적인 시뮬레이션을 제공하지만, 방대한 데이터와 복잡한 UI, 유료 결제 및 높은 하드웨어 사양으로 인해 일반 팬들이 가볍게 즐기기에는 진입장벽이 너무 높습니다.

**K-Tactics Lab 2026**은 이 간극을 메우는 도구입니다. 언론 보도를 통해 검증된 3대 전술 쟁점을 누구나 접근 가능한 웹 브라우저 상에서 직관적인 인터랙션으로 해결하고, 그 선택에 따른 상충 관계(Trade-off)와 대중 여론의 반응을 즉각적으로 체감할 수 있도록 설계했습니다.

## 🛠️ 3. Tech Stack (기술 스택)

| Component | Choice | Why (선택 이유) |
| :--- | :--- | :--- |
| **Core Framework** | `Vanilla HTML5 / CSS3 / JS (ES6+)` | 무거운 프레임워크나 빌드(`npm install`) 종속성 없이, 모바일/PC 브라우저에서 **1초 즉각 로딩** 및 심사위원 접근성 극대화 |
| **UI / Styling** | `Vanilla CSS Glassmorphism` | 스포츠 전술판 특유의 고급스러운 글래스모피즘 및 화면 진동(`Shake`), 테두리 발광(`Glow`) 등 동적 60fps 애니메이션 구현 |
| **Image Export** | `html2canvas v1.4.1` | 최종 시뮬레이션 명함 카드를 클라이언트 단에서 2배수 고화질 PNG 이미지로 캡처하여 소셜 미디어 바이럴 유도 |
| **Hosting / Deploy** | `Vercel / GitHub Pages` | 백엔드 서버 인프라 관리 없이 CDN을 통한 전 세계 무료 정적 웹 배포 및 **월 운영 비용 $0** 달성 |

## ✨ 4. Key Features (핵심 기능)

1. **👆 디바이스 이원화 스쿼드 전술판 (Hybrid Squad Builder)**
   * PC에서는 직관적인 **마우스 드래그 앤 드롭(Drag & Drop)**으로, 모바일에서는 **클릭 맞교환(Click-to-Swap)**으로 구장 선수와 벤치 멤버를 자유롭게 교체합니다.
2. **🤖 AI 수석 코치 '차비브(Cha-Vibe)'의 실시간 팩트폭력**
   * 유저가 공격수(손흥민)를 수비수나 골키퍼 위치에 배치하는 등 비정상적인 전술 기행을 벌이면, 즉각 **[비상 경보]**를 울리며 여론 지지율을 폭락시킵니다.
3. **🔥 다이내믹 Vibe Meter & 네티즌 중계창 스트리밍**
   * 팬 여론 지지도(0~100%)에 따라 화면 전체가 지진 난 듯 흔들리는 경고 효과가 작동하며, 하단 중계창에서는 4.5초마다 현재 여론 상태에 매칭되는 네티즌 댓글이 실시간 스트리밍됩니다.
4. **🏆 3단계 매치 시뮬레이션 & 바이럴 명함 캡처**
   * 조별리그 1~3차전 가상 매치 애니메이션 중계 후, 내 전술 성적표를 고화질 PNG 명함 이미지로 다운로드하여 SNS에 인증할 수 있습니다.

## 🧠 5. Key Design Decisions (핵심 설계 결정)

1. **인터랙션 구조: 단일 드래그 앤 드롭 → 디바이스 이원화(Hybrid) 설계**
   * *이유:* 기본 HTML5 Drag & Drop API는 iOS Safari 및 Android Chrome 터치 환경에서 제약이 큽니다. 모바일 심사위원 및 사용자들의 호환성을 100% 보장하기 위해, PC 환경은 드래그 앤 드롭, 모바일 환경은 클릭 기반 교체(Click-to-Swap)로 인터랙션을 이원화했습니다.
2. **밸런스 엔진: 무거운 매치 AI → 경량화 게임 밸런스 보정 알고리즘**
   * *이유:* 복잡한 물리/예측 엔진은 개발 대비 대중이 인과관계를 즉각 파악하기 어렵습니다. 포메이션 기본값에 전술 지침 및 풀백 역할의 보정값($$\Delta$$)이 선형 합산되고 상충(Trade-off)되는 모델을 채택하여, 사용자가 내린 선택의 피드백을 직관적으로 체감할 수 있게 했습니다.
3. **바이럴 전략: 일방적 링크 공유 → 명함 이미지 추출 메커니즘**
   * *이유:* 단순 URL 공유는 텍스트 위주라 클릭 유인 효과가 덜합니다. 유저의 소셜 미디어 인증 욕구를 자극하기 위해, 최종 성적과 '명장 지략가' 타이틀이 적힌 커스텀 명함을 고화질 PNG 파일로 제공하는 바이럴 촉진 메커니즘을 설계했습니다.

## 🚀 7. Getting Started (실행 및 배포 안내)

본 프로젝트는 복잡한 사전에 필요한 인프라나 환경 설정이 전혀 없습니다. (No prerequisites, No Node.js build required!)

### 💻 Local Run (로컬 실행)
1. 본 저장소를 다운로드(ZIP) 혹은 `git clone` 합니다.
```bash
git clone https://github.com/your-username/k-tactics-lab.git
cd k-tactics-lab
```
2. 폴더 내의 **`index.html`** 파일을 더블 클릭하여 웹 브라우저(Chrome, Safari, Edge 등)로 열면 즉시 실행됩니다.
