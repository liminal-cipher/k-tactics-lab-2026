## 📌 1. 서비스 개요 (Service Overview)

**K-Tactics Lab 2026**은 대한민국 축구 국가대표팀이 겪고 있는 고질적인 전술적 문제점들을 축구 팬(유저)이 직접 '국가대표 감독'이 되어 해결해 보는 **인터랙티브 웹 전술 시뮬레이션 서비스**입니다.

축구를 사랑하지만 복잡한 전술 전문가는 아닌 일반 팬들도 마우스 드래그 한 번으로 쉽게 전술을 짜고, 실시간 네티즌 여론 지지도(Vibe Meter)와 AI 수석 코치의 돌직구 피드백을 통해 **"내가 직접 대표팀을 구원한다!"**는 카타르시스와 재미를 느낄 수 있도록 설계되었습니다.

---

## 🎯 2. "감독 경험" 설계 의도 (Manager Experience Design)

한국 축구 팬들이 가장 답답해하는 **3대 핵심 논란/과제**를 게임적 인터랙션으로 풀어냈습니다:
1. **🚫 U자형 빌드업 극복:** 의미 없이 공을 뒤로만 돌리는 백패스를 금지하고, 이강인의 하프스페이스 강제 침투 루트를 직접 지시합니다.
2. **🛡️ 측면 자동문 풀백 수비 해결:** 인버티드 풀백, 쓰리백 스토퍼, 오버래핑 윙백 등 역할(Role)을 부여하여 팀 밸런스를 맞춥니다.
3. **🔄 후반 60분 '플랜 B' 조커 투입:** 특정 스타 선수에만 의존하는 '해줘 축구'를 탈피하고, 오현규·배준호·양민혁 등 체력과 스타일이 다른 조커 카드를 선택해 극장골 승부를 띄웁니다.

## 🖥️ 3. 핵심 기능 및 인터랙션 명세 (Core Features)

### ① 👆 하이브리드 전술판 (Drag & Drop + Role Setup)
* **드래그 앤 드롭:** 구장 위 선수들을 끌어서 자유롭게 위치를 변경하거나, 벤치 멤버(이동경, 김문환, 송범근 등)와 즉시 교체할 수 있습니다.
* **클릭 팝업 모달:** 선수를 클릭하면 전술 임무(Role)를 변경할 수 있는 모달이 작동합니다.

### ② 🤖 AI 수석 코치 '차비브(Cha-Vibe)'의 실시간 팩트폭력
* 전술 변경 시 체계적인 분석을 제공하며, 선수를 비정상적인 위치(예: 공격수 손흥민을 최후방 수비수/골키퍼로 배치)로 드래그할 경우 즉각 **[비상 경보]**를 울리며 여론 지지도를 폭락시킵니다.

### ③ 🔥 다이내믹 Vibe Meter & 실시간 네티즌 중계창
* 팬 여론 지지도(0~100%)가 30% 이하로 추락하면 **화면 전체가 흔들리는 지진(Shake) 효과와 붉은 경고등**이 점멸하며, 80%를 돌파하면 축하 글로우 효과가 나타납니다.
* 바 하단에는 실제 라이브 축구 중계창처럼 네티즌들의 반응이 4.5초마다 실시간 스트리밍되어 현장감을 극대화합니다.

### ④ 🏆 3단계 매치 시뮬레이션 & 명함 카드 PNG 다운로드
* 시뮬레이션 가동 시 조별리그 1~3차전의 극적인 경기 중계 텍스트가 애니메이션으로 전개된 후 최종 감독 성적표가 등장합니다.
* **`html2canvas`** 엔진을 탑재하여 클릭 한 번으로 내 전술 명함 카드를 **고화질 PNG 이미지로 저장**하고 SNS에 공유할 수 있습니다.

## 🛠️ 4. 사용 기술 (Tech Stack)

* **Core:** `Vanilla HTML5`, `Vanilla JavaScript (ES6+)` (프레임워크 종속성 없이 압도적인 로딩 속도 구현)
* **Styling:** `Vanilla CSS3` (Glassmorphism, CSS Grid/Flexbox, Dynamic Keyframe Animations, Dark Mode Sports Palette)
* **Libraries:** `html2canvas v1.4.1` (결과 명함 카드 이미지 캡처/다운로드용 CDN)

## 🚀 5. 프로젝트 실행 방법 (How to Run Locally)

본 프로젝트는 복잡한 `npm install`이나 백엔드 서버 구동, API 발급이 **전혀 필요 없는 순수 정적 웹앱(Static Web App)**입니다! 심사위원 및 사용자 누구나 1초 만에 실행할 수 있습니다.

### 방법 A: 브라우저에서 직접 실행 (가장 간편한 방법)
1. 본 저장소(Repository)를 다운로드(ZIP) 또는 `git clone` 합니다.
2. 폴더 내의 **`index.html`** 파일을 더블 클릭하여 웹 브라우저(Chrome, Edge, Safari 등)로 엽니다.
3. 즉시 플레이 가능! 🎉

### 방법 B: 로컬 개발 서버 실행 (VS Code Live Server)
1. Visual Studio Code에서 프로젝트 폴더를 엽니다.
2. `Live Server` 확장 프로그램을 통해 `index.html`을 구동합니다.
